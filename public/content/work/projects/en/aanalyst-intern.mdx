## Overview

As a Data Analyst Intern, responsibility was held for analyzing data and providing insights to help make informed decisions.

## Key Features

- **Data Analysis**: Data was analyzed and insights were provided to help make informed decisions.
- **Data Visualization**: Visualizations were created to help understand data trends and patterns.
- **Data Modeling**: Data models were developed and data analysis was performed to identify patterns and trends.
- **Data Integration**: A shared design library in Figma was set up in close collaboration with designers. This library was synchronized with the codebase, ensuring design handoffs were seamless and that design tokens remained consistent across both platforms.
- **Documentation and Usage Guidelines**: Comprehensive documentation with Storybook was developed to showcase components, usage patterns, and best practices, ensuring the design system is easy to adopt by other teams.

## Technologies Used

- **Excel**: For data analysis and visualization.
- **Python**: For data analysis and visualization.
- **SnowFlake**: For data analysis and visualization.
- **Tableau**: For data analysis and visualization.



## Challenges and Learnings

As this was a first time working with Python for data analysis and visualization, a steep learning curve was faced. The project required new libraries and frameworks, such as Pandas and Matplotlib, to be quickly learned and applied to real-world data.

One of the biggest challenges faced was dealing with the quality of the data. The data was raw and had not been cleaned or processed, which made it difficult to work with. A pipeline using Python was developed to clean and process the data, which was a great learning experience.

Another challenge was visualizing the data in a meaningful way. Different visualization tools and techniques had to be experimented with to find the best way to communicate the insights gained from the data.

Overall, this project was a great learning experience. A lot was learned about data analysis and visualization, and experience was gained working with new technologies and tools. A deeper appreciation was also gained for the importance of data quality and the challenges of working with real-world data.
