## Overview

As a Data Analyst Intern, responsibility was held for analyzing data and providing insights to help make informed decisions. 





## Key Features

- **Data Analysis**: Data was analyzed and insights were provided to help make informed decisions.
- **Data Visualization**: Visualizations were created to help understand data trends and patterns.
- **Data Modeling**: Data models were developed and data analysis was performed to identify patterns and trends.
- **Data Integration**: A shared design library in Figma was set up in close collaboration with designers. This library was synchronized with the codebase, ensuring design handoffs were seamless and that design tokens remained consistent across both platforms.
- **Documentation and Usage Guidelines**: Comprehensive documentation with Storybook was developed to showcase components, usage patterns, and best practices, ensuring the design system is easy to adopt by other teams.

## Technologies Used

- **Excel**: For data analysis and visualization.
- **Python**: For data analysis and visualization.
- **SnowFlake**: For data analysis and visualization.
- **Tableau**: For data analysis and visualization.




## Challenges and Learnings

- Entering a new domain of data analysis was a challenge in itself, as new tools and techniques had to be learned to work with large datasets.
- Dealing with nearly 50k data points was a challenge, as efficient methods had to be developed to process and analyze the data without sacrificing accuracy.

- Experience was gained working with large datasets, including data cleaning, processing, and analysis.
- Experience was gained using data visualization tools like Tableau to communicate insights effectively.
- Experience was gained collaborating with designers to develop a shared design library in Figma.
